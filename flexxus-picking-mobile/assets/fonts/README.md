# Custom Fonts Directory

This directory contains custom font files for the Flexxus Picking Mobile app.

## Required Fonts

Download the following font files from Google Fonts and place them in this directory:

### IBM Plex Sans (Body Font)
- IBM Plex Sans-Regular.ttf
- IBM Plex Sans-Medium.ttf
- IBM Plex Sans-SemiBold.ttf

Download from: https://fonts.google.com/specimen/IBM+Plex+Sans

### Barlow Condensed (Display/Headings Font)
- Barlow Condensed-Regular.ttf
- Barlow Condensed-SemiBold.ttf
- Barlow Condensed-Bold.ttf

Download from: https://fonts.google.com/specimen/Barlow+Condensed

### IBM Plex Mono (Monospace Font)
- IBM Plex Mono-Regular.ttf

Download from: https://fonts.google.com/specimen/IBM+Plex+Mono

## Installation Steps

1. Visit each Google Fonts link above
2. Click "Download family" or select individual weights
3. Extract the downloaded ZIP files
4. Copy the .ttf files listed above into this directory
5. The files should be named exactly as listed (with spaces and capitalization)
6. Restart the Expo development server

## File Sizes Expected

- IBM Plex Sans-Regular.ttf: ~100-150 KB
- IBM Plex Sans-Medium.ttf: ~100-150 KB
- IBM Plex Sans-SemiBold.ttf: ~100-150 KB
- Barlow Condensed-Regular.ttf: ~80-120 KB
- Barlow Condensed-SemiBold.ttf: ~80-120 KB
- Barlow Condensed-Bold.ttf: ~80-120 KB
- IBM Plex Mono-Regular.ttf: ~100-150 KB

Total expected size: ~550-750 KB

## Placeholder Files

The current .ttf files in this directory are placeholders. Replace them with the actual font files downloaded from Google Fonts.
