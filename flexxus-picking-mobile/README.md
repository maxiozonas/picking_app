# Flexxus Picking Mobile

App móvil React Native (Expo) para operarios de depósito.

## Stack
- Expo SDK 55
- React Navigation
- TypeScript
- Zustand (state)
- Axios (HTTP)
- AsyncStorage (cache)

## Instalación
```bash
npm install
npx expo start
```

## Configuración API
El backend debe estar corriendo en http://localhost:8000

## Development
- iOS: Press 'i' in terminal
- Android: Press 'a' in terminal
