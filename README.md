# Flexxus Picking App - Workspace

Workspace que contiene backend y mobile de la aplicación de armado de pedidos.

## Estructura
\`\`\`
proyectos/
├── flexxus-picking-backend/    # Laravel API
├── flexxus-picking-mobile/     # React Native (Expo)
├── docs/                       # Documentación compartida
└── README.md                   # Este archivo
\`\`\`

## Repositorios
- Backend: [flexxus-picking-backend](./flexxus-picking-backend/)
- Mobile: [flexxus-picking-mobile](./flexxus-picking-mobile/)

## Desarrollo Concurrente
Ambos proyectos se desarrollan en paralelo:
1. Backend: \`cd flexxus-picking-backend && php artisan serve\`
2. Mobile: \`cd flexxus-picking-mobile && npx expo start\`
